# Course Review

This repository contains my personal course schedule, course notes, and collected syllabi for selected CUHK(SZ) courses related to quantitative finance.

The comments are highly subjective and are provided for reference only. Course content, instructors, grading policies, and workload may vary substantially across semesters. Please check SIS and official course pages for the most accurate and up-to-date information.

## Files

- `tex/editor_course_schedule.pdf`: the main course review PDF.
- `tex/editor_course_schedule.tex`: LaTeX source for the PDF.
- `syllabus/`: collected course syllabi. Course codes in the PDF link to the corresponding syllabus when available.

## Notes

- `WL` stands for workload and is a subjective estimate.
- The color of each course code indicates its broad area:
  - Green: Finance & Economics
  - Blue: Mathematics & Statistics
  - Red: Computer Science & Data Science
